DVDs Availability API

This deployment exposes zero-config API endpoints for Deer Valley Driving School. Use the links below to verify they respond in production.

  • GET /api/ping – Health check returning {"ok": true}
  • GET /api/zip-route – ZIP router returning city calendar data and the resolved calendar ID
  • GET /api/calendars – Live calendars API (`?account=parents` for the alternate account)
  • GET /api/availability – Dual-account Acuity availability proxy (defaults `date` to today in America/Phoenix)

Refer to the README for more details.
